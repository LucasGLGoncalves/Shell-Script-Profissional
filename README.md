# Shell Script Studies

This repository contains different Shell scripts developed during my studies of the book **Shell Script Profissional**, by **Aurélio Marinho Jargas**.

The purpose of this repository is to serve as a practical learning space where I can organize examples, tests, exercises, notes, and small projects created throughout my reading journey. The main goal is to practice the concepts presented in the book and gradually improve my ability to write more organized, reusable, and professional Shell scripts.

## Objective

The objective of this repository is to track my progress while studying Shell Script, practicing topics such as:

- file and directory manipulation;
- variables;

- loops;
- functions;
- argument handling;

- task automation;

## How to Run the Scripts

To execute a script, first give it execution permission:

```bash
chmod +x script-name.sh
```

Then run the file:

```bash
./script-name.sh
```

You can also run it directly with the interpreter:

```bash
bash script-name.sh
```

or, depending on the script:

```bash
sh script-name.sh
```
